С включением по умолчанию десктоп оболочки Unity в Ubuntu, необходимость в таких приложениях как Dash, которые осуществляют поиск похожих по написанию файлов через ввод с клавиатуры, практически отпала. Тем не менее существует ряд достаточно интересных приложения, одно из которых сейчас и рассмотрим.

Synapse – семантический лаунчер подобный Gnome Do или Kupfer, в основе работы которого используется Zeitgeist. Среди основных характеристик программы стоит отметить поддержку множества различных плагинов: для плееров Rhythmbox или Banshee, соединения с SSH хостами, загрузки картинок на фотохостинг Imgur, поиска в Интернете, словаря и многое другое.

[![Synapse](img/2011/09/20/19-00/synapse-6166104659-o.jpg)](img/2011/09/20/19-00/synapse-6166104659-o.jpg)

Для запуска используются горячие клавиши <kbd>Ctrl</kbd> + <kbd>Space</kbd>, которые при желание можно заменить в настройках на любую другую понравившуюся комбинацию. После запуска и ввода названия интересующего приложения, будут выводиться все похожие по написанию результаты, отсортированные по частоте использования. Если нажать клавишу <kbd>TAB</kbd> на выделенном файле, из ранее созданной выборки, то будет предложен ряд следующих опций: запуска, выполнения в терминале, загрузка в Imgur и т.п.

[![Synapse](img/2011/09/20/19-00/synapse-6166104857-o.jpg)](img/2011/09/20/19-00/synapse-6166104857-o.jpg)

## Что нового

Рассмотрим, что было добавлено в текущую стабильную версию Synapse 2.8: включена новая тема «Side Doish» с полностью переработанным пользовательских интерфейсом, новый плагин выводящий похожие файлы (Related Files), и новая категория Places.

[![Synapse в ubuntu](img/2011/09/20/19-00/synapse2-6166640688-o.jpg)](img/2011/09/20/19-00/synapse2-6166640688-o.jpg)

Другими изменениями в Synapse 0.2.8 стало: улучшение скорости вывода результатов поиска и времени запуска, оптимизирован поиск по Zeitgeist и исправлены ряд ошибок связанные с методом ввода IBus и SSH плагином.

## Установка Synapse в Ubuntu

Synapse 0.2.8 пока не работает с GTK 3, но хорошей новостью является то, что разработчики обещают реализоваться поддержку к следующей версии Synapse 0.3.0, поэтому установка доступна только под Ubuntu 10.04, 10.10 или 11.04:

```

sudo add-apt-repository ppa:synapse-core/ppa 
sudo apt-get update 
sudo apt-get install synapse
```
